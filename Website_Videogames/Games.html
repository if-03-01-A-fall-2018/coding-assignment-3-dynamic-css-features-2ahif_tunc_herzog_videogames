Tetris

You control the bricks falling down. Fit them in a row to get points.

Use the arrow buttons.

The score will be reset automatically if the bricks stack up all the way.

Snake

Use the arrow keys to move the snake around the board. The snake "eats" the red blocks and thereby grows larger. The game ends when the snake moves into itself.

The goal is to make the snake as large as possible before that happens.
